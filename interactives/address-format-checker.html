Address Format Checker

Learn to identify which blockchain an address belongs to

Common Address Formats:

0x...Ethereum & EVM chains (Polygon, Arbitrum, etc.) - 42 chars
1...Bitcoin (Legacy) - 26-35 chars
3...Bitcoin (P2SH) - 26-35 chars
bc1...Bitcoin (SegWit) - 42-62 chars
VariousSolana - 44 chars, Base58

Important

Always verify the address format matches the network you're sending to! Sending ETH to a Bitcoin address (or vice versa) will result in permanent loss.
